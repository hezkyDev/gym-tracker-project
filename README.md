# 4-Day Gym Tracker Pro

Phone-friendly gym tracker split into systematic project files.

## Structure

- `index.html` — main app page
- `css/styles.css` — design and layout
- `js/app.js` — workout plan, charts, history, import/export
- `assets/` — icons/images
- `backups/` — optional exported JSON backups
- `docs/HOW_TO_USE.md` — simple guide
