# Gym Tracker Project Guide

## Folder structure

```text
gym-tracker-project/
├── index.html
├── css/styles.css
├── js/app.js
├── assets/
├── backups/
└── docs/HOW_TO_USE.md
```

## How to run

Open `index.html` in a browser.

## How to edit

- Workout plan, weekly targets, and app logic: `js/app.js`
- Colors, spacing, and layout: `css/styles.css`
- Page sections and navigation: `index.html`

## iPhone / GitHub Pages

Upload the contents of this folder to your GitHub repository. Keep the main file named `index.html`.

## Backup

The app saves data in browser `localStorage`. Use **Export Data** in the app to save a `.json` backup to iCloud Drive.
